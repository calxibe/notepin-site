v1.0.1 January 28, 2026

‚ú® New Features

  • RSS/Atom Feed Support ‚Äî New file format support for .rss and .atom files with syntax highlighting, auto-formatting, code folding for XML elements, clickable URLs (with entity decoding), and feed statistics (tags, attributes, namespaces, etc.).
  • Code Folding (JS) ‚Äî Collapse/expand functions, classes, objects, and code blocks. Hover over a foldable line to see the fold toggle. Use Collapse All / Expand All buttons in the toolbar. Can be enabled/disabled in settings.

  • Code Folding Setting (JSON) ‚Äî Added a setting to enable/disable the collapsible JSON feature.
  • Code Statistics Panel ‚Äî View file size, line count, and character count. Language-specific stats: JSON (object depth, key count, array sizes), JS (function count, variable count), CSS (selector count, rule count, media queries).
  • Hex/RGB Color Preview (CSS) ‚Äî Color swatches appear next to color values. Click to copy color in different formats. Includes a color picker.
  • Breadcrumb Navigation (JSON) ‚Äî Shows path like root ‚Ä∫ data ‚Ä∫ users[0] ‚Ä∫ name when hovering/clicking. Click any breadcrumb to jump to that level.
  • Clickable URLs & Imports ‚Äî URLs in strings and import/require paths are now clickable links.
